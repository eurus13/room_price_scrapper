# room_price_scrapper
Room price scrapper for CraigsList

CD to
```/cl/spiders```

run from term with 
```scrapy crawl rent -o results.csv```

## URL to set is scraper start w/:
```start_urls = ['https://philadelphia.craigslist.org/search/apa']```


## Results soted in results.csv in 
```/cl/spiders```
